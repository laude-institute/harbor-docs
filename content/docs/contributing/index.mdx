Please consider contributing to Harbor!

## Reporting issues

If you find a bug or have a feature request, please open an issue on the [GitHub repository](https://github.com/laude-institute/harbor/issues).

## Contributing code

If you want to contribute code to Harbor, please open a pull request on the [GitHub repository](https://github.com/laude-institute/harbor/pulls).

## Discord

Even if you don't want to contribute code, you can still [join our Discord server](https://discord.gg/6xWPKhGDbA) to discuss ideas, ask questions, and get help.
